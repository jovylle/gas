Quick insights

Charts & Iran war baseline

GlobalPetrolPrices publishes percent changes since Feb 23, 2026 (their last weekly point before the Iran war). We scrape that table to compute implied pre-war prices in the same currency as today: price ÷ (1 + %/100). Current prices come from OpenVan. OpenVan does not expose historical retail prices in the API; the line chart uses data/history.json: daily snapshots from this repo, an implied Feb 23 point (first run only), and past data/fuel.json versions from git when full history is available.
