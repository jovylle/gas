Fuel prices worldwide

Retail gasoline and diesel by country. This page loads data/fuel.json from the repo, then merges live OpenVan in your browser for any country whose source-quote date is newer (or the same calendar day) than the repo copy—check the live tag on rows. Prices are shown in PHP for quick cross-country comparison. No app server required.

Quick insights

Charts & Iran war baseline

GlobalPetrolPrices publishes percent changes since Feb 23, 2026 (their last weekly point before the Iran war). We scrape that table to compute implied pre-war prices in the same currency as today: price ÷ (1 + %/100). Current prices come from OpenVan. OpenVan does not expose historical retail prices in the API; the line chart uses data/history.json: daily snapshots from this repo, an implied Feb 23 point (first run only), and past data/fuel.json versions from git when full history is available. Current bar values use the same merged prices as the table (live OpenVan when newer than the repo snapshot). All displayed values are converted to PHP (via ECB/Frankfurter cross rates) for apples-to-apples comparisons.
